ray-mmd内置天空盒调整参数简介

2023-06-05

完成ray渲染的基本流程后，就要开始调各种参数了，本篇介绍ray渲染内置的几个天空盒的参数

在模型操作栏中，选择导入的天空盒，在表情栏即可进行其相关参数调整。

以下介绍中，把表情四栏按从左到右，从上到下分成 第一栏、第二栏、第三栏、第四栏。

Helipad GoldenHour #

第一栏 #

  • Balance 可以看到该栏中有BalanceR+-/G+-/B+-以及Gray+的调整，它们即对应色调红绿蓝（RGB）三色以及灰色的调整。 例如，调高BalanceR+栏，画面将偏红色

第二栏 #

  • EnvDiffLight 该参数为天空盒的光源强度，调高该参数，天空盒对模型的影响将会变亮
  • EnvSpecLight 该参数为漫反射强度，调高该参数，模型漫反射增强

第三栏 #

  • Diffuse 该栏调整漫反射色调（RGB&Gray）

第四栏 #

  • Specular 该栏调整镜面反射色调（与Diffuse同理）

Sky Hemisphere #

该天空盒初始为纯白背景，一般适用于实景合成/唯美风渲染

第一栏 #

  • Top 可以看到该栏中有TopH+/S+/V+-，它们调整该天空盒的顶部色调效果

第二栏 #

  • Medium 与第一栏同理，调整中部色调

第三栏 #

  • Bottom 与上同理，调整底部色调

第四栏 #

  • EnvDiffLight 该参数为天空盒的光源强度，调高该参数，天空盒对模型的影响将会变亮
  • EnvSpecLight 该参数为漫反射强度，调高该参数，模型漫反射增强

Sky Night #

该天空盒与Sky Hemisphere几乎一样，区别是该天空盒带有夜晚的月亮和流星效果

Time of day #

第一栏 #

  • SunRadius 调整太阳半径大小

第二栏 #

  • EnvDiffLight 该参数为天空盒的光源强度，调高该参数，天空盒对模型的影响将会变亮
  • EnvSpecLight 该参数为漫反射强度，调高该参数，模型漫反射增强

第三栏 #

  • MiePhase 该参数可以影响丁达尔效应的强弱
  • FogRange 调整雾气距离
  • FogIntensity 该参数可以影响雾气效果强度

第四栏 #

  • Cloud 云的密度，调高该参数，云将增多
  • CloudSpeed 云的流动速度

Time of night #

该天空盒具有夜间星空银河与流星效果，参数调整与Time of day基本一致。 区别是该天空盒第四栏的参数为影响星星的效果
