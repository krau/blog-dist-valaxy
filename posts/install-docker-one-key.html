注意是 docker compose 而不是 docker-compose. 执行命令时候也没有这个杠

sudo groupadd docker # 建立docker用户组
sudo usermod -aG docker $USER #将当前用户加入 docker 组

安装docker-compose #

执行以下命令

curl -L https://github.com/docker/compose/releases/download/v2.14.0/docker-compose-linux-`uname -m` > ./docker-compose

注释：

  • uname -m用于获取本机架构
  • 注意将/2.14.0/换成所需的版本号，可前往docker-compose的仓库查看详情

记得授予权限 chmod +x ./docker-compose 移动到正确路径 mv ./docker-compose /usr/local/bin/docker-compose
